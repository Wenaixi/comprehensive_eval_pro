# Security Policy

本项目以安全与合规为前提开展学习研究实践。

## Supported Versions

Only the latest version in this repository is supported.

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this project:

- Do not publish exploit details.
- Open an issue with a minimal reproduction.
- Do not include any real credentials, tokens, or sensitive user data.

We will respond as quickly as possible.

如需私下沟通（例如误提交敏感信息紧急处理）：cep@wenxi.dev
